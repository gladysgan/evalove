UI DESIGN SYSTEM / RECORD 1.0

状态色规范

Evalove 的视觉语言向 Ant Design 的企业后台靠拢：白底、轻边框、清晰层级和稳定状态色。主题蓝 #2673DD 用来承载主操作，状态色只用于成功、错误、默认、处理中和警告。

设计原则

01证据优先：状态、输出、review、attempt 都要能追溯。
02克制高亮：#2673DD 只用于主按钮、当前导航和可点击入口；状态不要混用主题色。
03轻量容器：8px radius、浅灰边框、柔和阴影，形成干净后台感。
04可扫读密度：数字用 serif 放大，标签用 mono 收敛，表格保持紧凑。

颜色 Token

白色作为主背景，#2673DD 作为主题色。状态色参考 Ant Design 语义：绿色成功、红色错误、灰色默认、蓝色处理中、橙色警告。

字体与排版

Display 使用系统可用 serif，制造“质检报告”质感；正文使用干净 sans；标签、编号和 token 使用 mono。

92%

用于 gate score、hero 标题、metric 数字。字号大、行高紧，承载页面记忆点。

Body / Sans

Skill Owner 创建 release eval，Reviewer 在队列中集中处理 comparison。正文保持清晰，避免装饰化。
